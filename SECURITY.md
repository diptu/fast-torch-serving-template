# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
privately rather than opening a public issue:

- Open a [GitHub Security Advisory](../../security/advisories/new) for this
  repository, or
- Contact the repository maintainer directly.

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any relevant logs or proof-of-concept code

We'll acknowledge reports within a few business days and follow up as the
issue is investigated and fixed.

## Supported Versions

This is a template repository with no formal release/version support
policy — security fixes are applied to the `main` branch only.
